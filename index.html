Introducción

Volovan Volo POS es un sistema de punto de venta diseñado para la operación diaria de una panadería.

El sistema permite administrar ventas, inventario, clientes, caja, reportes, usuarios y tickets desde una sola plataforma.

  • Control de ventas y stock.
  • Administración de caja.
  • Control de clientes y créditos.
  • Reportes y auditoría.

Requisitos mínimos

  • Java 21 compatible con JavaFX.
  • MySQL instalado.
  • Resolución mínima 1366x768.
  • Impresora térmica 58mm u 80mm.

Inicio de sesión

El usuario debe ingresar su usuario y contraseña para acceder al sistema.

  • Abrir Volovan Volo POS.
  • Ingresar usuario y contraseña.
  • Presionar "Iniciar Sesión".

Problemas comunes

  • Error de conexión.
  • Error de impresora.
  • Stock insuficiente.
  • Login incorrecto.

Buenas prácticas

  • No compartir cuentas.
  • Respaldar base de datos.
  • Cerrar caja diariamente.
  • Revisar stock constantemente.
